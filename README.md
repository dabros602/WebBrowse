# WebBrowse
## What is WebBrowse?
WebBrowse is a simple browser based off the Tkhtml3 framework. 
## Features of WebBrowse
- HTML and CSS
- Forward and back buttons
- Home button (optional)
- Refresh button
- Settings file
## Requirments
- tkinterweb
- tkinterweb-tkhtml
- tkinterweb-tkhtml-extras
Use
```
pip install requriments.txt
```
to install all requirments automatically
## Future Plans
